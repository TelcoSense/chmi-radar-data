# chmi-radar-data

Radar data scraping tools and a simple radar data image server.

## Overview

This repository contains tools for downloading and processing radar data provided by the Czech Hydrometeorological Institute, along with a lightweight image server for serving preprocessed radar products. The project is intended for research, experimentation, and integration into larger meteorological or hydrological analysis pipelines.

## Contents

- Radar data scraping and download utilities
- Basic preprocessing and file organization
- Simple image-serving backend for radar products
- Supporting scripts and helpers

## Usage

Typical workflows include scheduled data downloads, preprocessing, and exposing resulting radar images via HTTP for visualization or further analysis.
